Hydra

The next-generation HTTP stream aggregator for Stremio. Rebuilt from the ground up to never fail.

View on GitHub Installation
🤖

AI Healing

Using Google Gemini and OpenAI as fallbacks, Hydra dynamically heals broken extractors by analyzing the DOM to scrape stream links natively. It never misses.

⚡

Intelligent Caching

Fully backed by Redis for centralized distributed caching. Stop hammering hosters and start streaming in milliseconds instead of seconds.

🚀

High Concurrency

Built with intelligent proxy rotation and dynamic retries. It laughs in the face of rate limits and extracts multiple links simultaneously.

Ready to deploy?

Hydra is open source. You can host it on your own infrastructure instantly using Docker.

docker run \ --detach=true \ --name hydra \ --rm \ --pull always \ --publish 51546:51546 \ --env REDIS_URL="redis://your-redis-host:6379" \ --volume /tmp:/tmp \ zainulnazir/hydra

After starting, visit http://localhost:51546/configure to enter your API keys and install to Stremio.
